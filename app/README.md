# ForgetMeNot Tray Shell

![ForgetMeNot flower logo](./icons/flower.png)

Source for `forgetmenot.exe` — the Windows tray shell for the ForgetMeNot bot runtime.

The name comes from the old German forget-me-not legend: a knight tried to pick the flower for his beloved, fell into the river under the weight of his armor, and called out "Vergiss mein nicht" as he threw the flowers back to shore. The bot uses that symbolism literally — it is built to remember chatters, stream lore, and inside jokes across streams.

The runtime (Node SEA from `services/forgetmenot/`) is **embedded** inside this binary at build time and extracted on first launch. End users get a single file.

## What it does

- Extracts the embedded runtime to `%LOCALAPPDATA%\ForgetMeNot\runtime\forgetmenot.exe` on first launch (re-extracts only if SHA-256 differs)
- Spawns and supervises the runtime as a child process (Windows job object: runtime is killed if the tray dies for any reason)
- Polls `/health` every 10s, updates tray icon color
- Right-click menu:
  - Status (color indicator + label)
  - Open Review Dashboard (opens toolkit in browser)
  - Pause / Resume Replies (in-memory, no config write)
  - Safe Mode toggle (proxies to auth worker)
  - Restart Runtime
  - Quit

Anything richer (logs, stats, settings) lives in [toolkit.deutschmark.online](https://toolkit.deutschmark.online/tools/chat-bot).

## Build

Requires Go 1.22+. Install from <https://go.dev/dl/>.

The runtime exe must exist at `../forgetmenot/build/forgetmenot.exe` first — build it with `cd ../forgetmenot && build-exe.bat`.

Then:

```
build.bat
```

Output: `forgetmenot.exe` (~93 MB — runtime is embedded inside).

## Icons

Two layers, regenerated automatically by `build.bat`:

1. **Status tray icons** (`healthy.ico`, `degraded.ico`, `error.ico`, `paused.ico`, `loading.ico`) — the colored flower shown in the notification area. Embedded into the Go binary via `//go:embed icons/<name>.ico`, so `go build` always re-reads whatever is on disk. Generated by `icons/generate.py` from `icons/flower.png`.
2. **Brand / app icon** (`forgetmenot.ico` → `rsrc_windows_amd64.syso`) — shown in File Explorer, taskbar, Alt-Tab. Embedded via a Windows resource object (`.syso`) that Go picks up automatically, but only if it's been regenerated from the current `.ico`. Built with `rsrc`.

`build.bat` runs `generate.py` (if Python + Pillow are present) and `rsrc` (if installed) before `go build`. Both are skippable — the committed `.ico` / `.syso` files are used if the tools are missing.

To edit the flower design, change `icons/flower.png` and re-run `build.bat`.

Prerequisites for icon regen:

- Python 3 + Pillow (`pip install Pillow`)
- rsrc (`go install github.com/akavel/rsrc@latest`)

## Logs

Runtime stdout/stderr is captured to `%LOCALAPPDATA%\ForgetMeNot\runtime.log`.

## Single instance

Uses a Windows named mutex (`Global\ForgetMeNotTraySingleInstance`). Launching a second instance exits silently.
